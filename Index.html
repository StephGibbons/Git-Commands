Commonly Used Git Commands

git config

Tells Git who you are using git config.

Example: git config --global user.email srgibbons03@gmail.com

git init

To create a new repository.

git add

Adds one (or more) files to staging

git commit

Commits files or changes to files.

git commit -a (this commits files that have been added uning "git add")

git commit -m (this commits changes)

git push

To send changes to the master branch of your repository.

git status

Provides status update on your repository listing the files you have changed and highlighting files that needed to be added or committed.

git checkout

Use to switch from one branch to another

git branch

tells you which branch you are in, as well as what other branches are in the repository.

git pull

Take and merge changes on the remote server
